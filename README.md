## Trying it out
If for some reason you wish to try it out:

1. run the migrations:
```bash
# from the project root
sqlite3 lib/db/sqlite.db < lib/db/migrations/init.sql
```
2. start local server
```bash
# from the project root
php -S localhost:8000
```
